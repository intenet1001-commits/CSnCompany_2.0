## Skill routing

When the user's request matches cs-youtube, ALWAYS invoke it using the Skill tool as your FIRST action.

Key routing rules:
- 유튜브 쇼츠, YouTube Shorts, 숏폼 만들어, 쇼츠 제작, 영상 만들어 → invoke cs-youtube
- /cs-youtube → invoke cs-youtube
- 대본 써줘 + 유튜브 → invoke cs-youtube
- 경쟁사 분석 + 영상 → invoke cs-youtube
