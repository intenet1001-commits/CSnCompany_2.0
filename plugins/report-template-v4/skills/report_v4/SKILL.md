## 4. 번호 체계 및 계층 구조

### 4.1 전체 계층 구조

```
I. 대항목 (로마숫자)
   1. 번호항목 (아라비아 숫자)
      • 불릿항목 (속이 찬 dot)
      (1) 하위번호항목 (괄호 숫자)
          - 하이픈 불릿항목
```

### 4.2 대항목 (Section Heading)

```
I. 추진 배경 및 목표
II. 핵심 과제
III. 상세 실행 방안
IV. 전략적 도약 및 성과 지표(KPI)
[별첨] 법인대상 세미나 운영 방안
```

## 5. 텍스트 강조 (Bold Keyword)

### 5.1 강조 방식

불릿 항목에서 핵심 키워드를 굵게 처리하여 가독성 향상:

```
• 패밀리오피스 지원의 성공적 완수를 통해 검증된 역량을 바탕으로...
  ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^ (굵게)
```

### 5.2 사용법

```python
add_dot_bullet_item(doc, '전체 내용', bold_keyword='굵게 처리할 앞부분')
add_dash_bullet_item(doc, '전체 내용', bold_keyword='굵게 처리할 앞부분')
```

### 5.3 강조 규칙

- **굵게**: 대항목 제목, 번호, 표 헤더, 핵심 키워드
- **일반**: 본문 내용, 설명 텍스트
- **작은따옴표**: 영문 용어, 기준명 (예: 'Complete Care')

## 6. 각주 (Footnote)

### 6.1 각주 형식

- 본문: 위첨자 번호 `1)` 형태
- 페이지 하단: `1) 각주 내용` 형태

### 6.2 사용법

```python
# 단락 생성 후 각주 삽입
p = doc.add_paragraph()
run = p.add_run('단체(협회)')
set_font(run, size=11, bold=True)
create_footnote(doc, p, '단체(협회): 산업통상자원부 산하 사단법인...')
run = p.add_run('를 대상으로 먼저 시행')
set_font(run, size=11)
```

### 6.3 각주 스타일

| 항목 | 값 |

| 폰트 | 나눔명조 |
| 본문 위첨자 | 9pt |
| 하단 각주 내용 | 9pt (18 half-points) |

## 8. 대항목 확장/축소 기능

### 8.1 기능 설명

Word 2016 이상에서 대항목(I, II, III 등) 왼쪽에 삼각형 아이콘이 표시되어 클릭 시 해당 섹션을 접거나 펼칠 수 있음.

### 8.2 구현 방식

```python
def set_collapsible_heading(paragraph, outline_level=1):
    pPr = paragraph._p.get_or_add_pPr()
    outlineLvl = OxmlElement('w:outlineLvl')
    outlineLvl.set(qn('w:val'), str(outline_level - 1))
    pPr.append(outlineLvl)
```

### 8.3 Outline Level

| Level | 용도 |
| --- | --- |
| 1 | 대항목 (I, II, III...) |
| 2 | 중항목 (필요시) |
| 3 | 소항목 (필요시) |

## 10. Python 함수 레퍼런스

### 10.1 기본 함수

| 함수 | 용도 | 주요 파라미터 |

### 10.2 문서 구조 함수

| 함수 | 용도 | 주요 파라미터 |

## Rules

1. **폰트 일관성**: 모든 텍스트는 나눔명조 사용
2. **줄간격 통일**: 1.15배 줄간격 유지
3. **계층 구조 준수**: I. → 1. → • → (1) → - 순서 유지
4. **키워드 강조**: 불릿 항목에서 핵심 키워드 앞부분 굵게 처리
5. **표 형식**: 헤더는 중앙정렬+굵게+회색배경, 내용은 좌측정렬
6. **문서 종료**: 반드시 [끝] 표시로 마무리
