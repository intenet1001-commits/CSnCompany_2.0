# /report Command

증권사/금융사 전략/기획 부서 전문 보고서를 작성합니다.

## Usage

```
/report              # 보고서 템플릿 가이드 및 작성 시작
/report [주제]       # 특정 주제로 보고서 작성
```

## Features

- 7단계 계층 구조 (I. → 1. → • → (1) → - → o → *)
- 나눔명조 폰트 기반 일관된 스타일
- python-docx 기반 DOCX 변환 지원
- 대항목 확장/축소 기능
- 각주 및 표 지원

## Output

마크다운 형식의 보고서와 DOCX 변환 가능한 구조화된 문서를 생성합니다.
